# Data Model for Phase 5: RAG with Qdrant

This document details the data model for the Qdrant Cloud vector database, specifically for the `physical_ai_textbook` collection.

## Qdrant Collection: `physical_ai_textbook`

Each point in the Qdrant collection will store the following information:

### Payload (Metadata) Schema

| Field Name | Type          | Description                                                    | Example Value                                  |
| :--------- | :------------ | :------------------------------------------------------------- | :--------------------------------------------- |
| `text`     | `str`         | The original text content of the chunk.                        | "Physical AI involves the embodiment of AI..." |
| `source`   | `str`         | The file path of the original Markdown document.               | `docs/module1/chapter-1.md`                    |
| `chapter`  | `str`         | The title or identifier of the chapter the chunk belongs to.   | "Introduction to Physical AI"                  |
| `section`  | `str`         | The heading or subsection title the chunk belongs to.          | "Historical Context"                           |

### Vector

| Field Name | Type          | Description                                                                 |
| :--------- | :------------ | :-------------------------------------------------------------------------- |
| `vector`   | `List[float]` | The embedding vector generated by OpenAI's `text-embedding-3-small` model. |
| **Size**   | `1536`        | The dimension of the embedding vector.                                      |
| **Metric** | `Cosine`      | The distance metric used for similarity search.                             |

## Example Qdrant Point Structure

```json
{
  "id": "uuid_or_int_id",
  "payload": {
    "text": "Physical AI (PAI) represents the tangible manifestation of artificial intelligence...",
    "source": "docs/module1/chapter-1.md",
    "chapter": "Chapter 1: Foundations of Physical AI",
    "section": "Introduction to Physical AI"
  },
  "vector": [0.123, 0.456, ..., 0.789] // 1536-dimensional vector
}
```
